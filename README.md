# vibe-skills - стартовый кит скиллов для AI-агентов

> **Ты ИИ-агент, которому дали эту ссылку со словами «настрой»?** Читай [AGENT-SETUP.md](AGENT-SETUP.md) и выполняй по шагам.

Курированный набор для работы с AI-агентами (Claude Code и Codex CLI): авторские скиллы,
проверенные в реальной работе, плюс каталог лучших публичных коллекций с навигацией, что брать и зачем.

## Что внутри

- **`skills/`** - авторские скиллы, готовые к установке:
  - `slide-craft` - ремесло слайда: композиция, сетка, ритм, обязательная петля «отрендери → посмотри глазами → поправь». Палитра - 10% результата, остальное здесь.
  - `data-analyst` - аналитика данных с жесткой дисциплиной против выдуманных цифр: каждая цифра прослеживается до первоисточника, расчеты кодом, а не «в уме».
  - `meeting-insights` - разбор транскриптов встреч и интервью: факты, боли, роли, противоречия; жесткое разделение «сказано / следует / гипотеза».
  - `strategic-advisor` - фреймворки решений: ТРИЗ, теория ограничений, первопринципы, инверсия, пре-мортем.
- **`SKILLS-LIST.md`** - поименный список всех скиллов по коллекциям (1599 уникальных имен).
- **`CATALOG.md`** - каталог проверенных публичных коллекций (Anthropic, Trail of Bits, Impeccable, Emil Kowalski, UI/UX Pro Max, ECC, Matt Pocock и др.) со ссылками и пометками, что в них золото.

## Быстрый старт

### Claude Code
```bash
git clone https://github.com/Yzyvan/vibe-skills.git
cp -r vibe-skills/skills/* ~/.claude/skills/
```

### Codex CLI
```bash
cp -r vibe-skills/skills/* ~/.codex/skills/
```

Скиллы - обычные папки с `SKILL.md` (инструкция + frontmatter), формат читают оба движка.

## Два совета, которые важнее любого скилла

1. **Заведи навигатор.** Файл `_INDEX.md` в папке скиллов: одна строка на скилл - «когда брать» и «что делает».
   Агент выбирает по нему, а не по памяти.
2. **Вшей агенту правило «Шаг 0».** В файл-вход агента (CLAUDE.md / AGENTS.md) добавь: перед любой задачей
   пробежать навигатор и НАЗВАТЬ пользователю выбранные скиллы одной строкой («Беру скиллы: X, Y»).
   Пользователь не должен помнить инструменты - это обязанность агента.

## Откуда это

Набор собран и обкатан в реальной работе: презентации, аналитика для консалтинга, кастомные
корпоративные системы, оркестрация нескольких агентов (Claude Code + Codex). Чужие скиллы здесь
не перепубликуются - в каталоге ссылки на авторов.
